### 재귀 함수

---

재귀 호출은 반드시 멈춰야 한다.

간단하게 해결할 수 있을 때만 사용한다.

```jsx
// 함수 선언문
function fact(n) {
  if (n <= 1) return 1;
  return n * fact(n - 1);
}
fact(5);
```

```jsx
// 함수 리터럴
var fact = function f(n) {
  if (n <= 1) return 1;
  return n * f(n - 1);
};
fact(5);
```

```jsx
// 익명 함수
var fact = function (n) {
  if (n <= 1) return 1;
  return n * arguments.callee(n - 1);
};
fact(5);
```
