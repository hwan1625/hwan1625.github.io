### 함수 정의

---

- 함수 선언문
  ```jsx
  function f(x) {
    return x;
  }
  ```
- 함수 리터럴
  ```jsx
  var f = function (x) {
    return x;
  };
  ```
- Function 생성자
  ```jsx
  var f = new Function("x", "return x");
  ```
- 화살표 함수 (ECMAScript 6)
  ```jsx
  var f = (x) => x;
  ```
