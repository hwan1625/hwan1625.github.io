### **함수 호출**

---

```jsx
var g = f(5);
```

```jsx
// 생성자 호출 (함수의 참조를 저장한 변수 앞에 new 키워드)
var obj = new Object();

// 메서드 호출 (객체의 프로퍼티에 저장된 값이 함수 타입)
obj.m = function() {...};
obj.m();
```

```jsx
// call, apply를 사용한 간접 호출
```

### 즉시 실행 함수

---

```jsx
(function(a,b) {...})(1,2);
(function() {...}());
+function() {...}()

var x = (function() {...})();
```
